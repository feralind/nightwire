# Nightwire

Offline dual-life crime hub RPG (True V0 + staged systems).

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — create a character, then use Crimes / Jobs / Education.

## Scripts

- `npm run dev` — local server
- `npm run build` — production build
- `npm test` — vitest (RNG/formulas/tick)

## V0 content floor

- 12 crimes / 8 jobs / 6 courses / 3 districts
- Dual ledger, heat/investigation, gym tracks, rival events
- Post-gate: Bazaar, Daily Ritual, Mastery stars
- V1: Power Tracks page

## Save

Persisted in `localStorage` key `nightwire-save-v1`. Export/import from Settings.
