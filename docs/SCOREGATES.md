# Nightwire scoregates (human verification checklist)

## Likeness (≥85)
- [ ] 3-row header + ticker persist across in-city nav
- [ ] ≥28 left-nav destinations visible
- [ ] Bordered modules + tables (not SaaS cards)
- [ ] Attempt → result modal → log
- [ ] Hospital/Jail/Travel takeover UI
- [ ] Classic density default

## UI Quality (≥90)
- [ ] Tokens only (no raw hex in pages)
- [ ] No layout shift on nav
- [ ] Interactive states present
- [ ] Tabular nums / aligned tables
- [ ] RequirementsBox on locked rows
- [ ] Empty/loading/error states
- [ ] No horizontal scrollbar at 1280×720

## 5-Minute Gate
- [ ] Playtester clicks Attempt ≥3 times in 5 minutes without prompting

## Hour-1 Gate
- [ ] Player can state next unlock within 15 minutes
- [ ] ≥2 progression tracks advanced
- [ ] Locked content clickable with requirements

## Hour-20 (post-gate / V1)
- [ ] Bazaar transaction
- [ ] Daily ritual completed
- [ ] Mastery level gained
- [ ] Power track investment
- [ ] Race entered or faction assisted
- [ ] Bounty hunted
